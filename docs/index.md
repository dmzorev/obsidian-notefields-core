# NoteFields Core documentation

NoteFields Core adds richer editors and reusable configuration to Obsidian properties while keeping values in standard YAML frontmatter.

## For users

- [Getting started](getting-started.html): install the plugin and configure your first field.
- [Fields and option collections](fields-and-options.html): Select, Multiselect, Nested, Icon, Color, and reusable options.
- [Obsidian Bases](bases.html): display and edit NoteFields values in Bases tables.
- [Data model and compatibility](data-and-compatibility.html): where NoteFields stores data and what remains portable.
